Quick Start Guide

Installation
------------
The ``data_morph`` package can be installed with ``pip``:

.. code:: console

   $ pip install data_morph

..
   You can also install with ``conda`` (coming soon):

   .. code:: console

      $ conda install data_morph -c conda-forge
